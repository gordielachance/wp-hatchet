== Description ==

Enables shortcodes to embed playlists, albums and single tracks from Hatchet.is, and other functions to communicate with the Hatchet.is API.

= Use shortcodes =

Use this code : [hatchet]HATCHET_URL[/hatchet], where HATCHET_URL is 
* a playlist url (eg.: https://hatchet.is/people/muesli/playlists/5095d5f87fb4401672fa194c_53d895852484f201a20117d0)
* an artist url (eg.: https://hatchet.is/music/Queen)
* an album url (eg.: https://hatchet.is/music/Queen/Greatest+Hits)
* a track url (eg.: https://hatchet.is/music/Queen/_/Don't+Try+So+Hard)

You can also use it like this :

* playlist : [hatchet username="USERNAME" hatchet_id="HATCHET-ID"][/hatchet] 
* artist : [hatchet artist="ARTIST"][/hatchet] 
* album : [hatchet artist="ARTIST" album="ALBUM"][/hatchet] 
* track : [hatchet artist="ARTIST" album="ALBUM" track="TITLE"][/hatchet] (album is optional)
